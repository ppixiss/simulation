# 🏝️ Pet project "Simulation" 🏄

Консольная реализация симуляции 2d-мира, написанная на Java версии 17.0.18 в стиле ООП.
Мир содержит акул (хищников), серферов (добычу), волны (ресурс для серферов) и статичные объекты.

### Дополнительная информация:

*   **Размер мира:** Поле 12х12 клеток.
*   **Скорость существ (по умолчанию):** Акула — 2 клетки за ход, серфер — 1 клетка за ход.
*   **Механика волн:** Когда серфер «ловит волну», его скорость увеличивается на 1 на следующий ход.
*   **Система атак:** Здоровье серфера — 100 единиц. Атака акулы наносит 50 единиц урона.

### Как запустить
1. Клонировать репозиторий
 ```bash
git clone https://github.com/ppixiss/simulation.git
cd simulation
 ```
2. Скомпилировать исходный код
 ```bash
javac -d out $(find src -name "*.java")
 ```

3. Запустить программу
```bash
java -cp out Main
```
